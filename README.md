## Project Setup

1. **Clone the repository**
```
git clone https://github.com/Thanisha23/AnyDo-responsive.git
```

2. **Navigate to the project directory and install dependencies for the project**
 ```
npm install
```
8. **Start the development server**
```
npm run dev
```
